Elemental ( Mágico )

 

 
 
Uma Build pouco utilizada, mas que tem seus benefícios. Sua distribuição é:
 
 

 

Força - Se baseia pelos itens que vai usar.
Inteligência - Coloque quantos pontos conseguir.
Destreza - Se baseia pelos itens que vai usar.
Constituição - 1000 para Mortal e Arch. 1000~2000 para Celestiais. (2000 pontos em celestiais level 220+)

 

 
 

 

Características:
Dano: Alto.
HP: Alto.
Defesa: Media.

 

 
 
 

 

Skills:

 

 
Pegue todas as skills de Elemental até a 8° e os buffs e passivas das outras classes de skills. De Natureza, pegue a transformação para Urso e Evocação pegue os Tigres.
 
 

 

Itens:

 

 
 

 

Set (Armadura): 
Mortal - Elmo 20/12 | Peito 30/10 | Calça 30/10 | Bota 24/10 | Luva 10/30
Arch - Arcana 50/70, 99/30, 120/30 | Peito 40/14 | Calça 40/14 | Bota 42/14 | Luva 14/70
Celestial - Arcana 120/30 | Peito 99/30 | Calça 99/30 | Bota 42/14 | Luva 14/70
 
Armas:
Mortal - Use Foice Platinada base 20~28 ataque magico.
Arch - Use Gleipnir base 36~84 ataque magico.
Celestial - Use Gae Bulg base 36~84 ataque magico.
 
Amuletos:
Use amuletos com base de HP, Defesa, Imunidade e Ataque mágico.
 
 
Natureza (Físico)
 
 
 
Uma das Build mais utilizadas no jogo. Possui muita defesa e um bom ataque físico. Sua distribuição é:
 
 
Força - Se baseia pelos itens que vai usar.
Inteligência - Se baseia pelos itens que vai usar.
Destreza - Coloque quantos pontos conseguir.
Constituição - 1000 para Mortal e Arch. 1000~2000 para Celestiais. (2000 pontos em celestiais level 220+)
 
 
Características:
Dano: Alto.
HP: Alto.
Defesa: Alta esquiva.
 
 
 
Skills:
 
Pegue todas as skills de Natureza até a 8° e os buffs e passivas das outras classes de skills.
 
 
 
Itens:
 
 
Set (Armadura): 
Mortal - Elmo 60/18 | Peito 30/7 | Calça 30/7 | Bota 30/18 | Luva 30/30
Arch - Arcana 50/70, 99/4.5, 120/120 | Peito 42/9 | Calça 42/9 | Bota 42/18 | Luva 18/70
Celestial - Arcana 120/120 | Peito 120/99/4.5 | Calça 120/99/4.5 | Bota 42/18 | Luva 18/70
 
Armas:
Mortal - Use duas Asas Draconianas base 45~72 ataque físico.
Arch - Use duas Balmungs base 81 | 81/3 | 90/3 | 99/3 | 126~240 ataque físico.
Celestial - Use duas Gaoths base 126~240 ataque físico.
 
Amuletos:
Brinco de Hercules Evento +11 | 2 ABS +9 30% HP/10% Crítico | 1 ABS 40 IMU/10% Crítico. Ou, use amuletos com base em HP, Imunidade, Defesa e Ataque físico.

Destreza - Coloque quantos pontos conseguir.
Constituição - 1000 para Mortal e Arch. 1000~2000 para Celestiais. (2000 pontos em celestiais level 220+)

Defesa: Alta esquiva.
